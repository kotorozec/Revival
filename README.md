## Build with:

-   Nextjs,
-   Tailwind and SASS,
-   Contenfull (Blog API),
-   SupaBase (Gallery Page),
-   NodeMailer (Forms),
-   In18 (Language version detection),
-   Framer Motion (Main animations and page transitions),

## Demo

[Revival Page](https://revival-nu.vercel.app/)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```
